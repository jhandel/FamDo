# FamDo - Family Dashboard

A touch-optimized family management dashboard for Home Assistant.

## Features

- **Family Members** - Create profiles with colors and avatars
- **Chore Management** - Assign chores, track completion, award points
- **Rewards System** - Kids earn rewards with their points
- **Todo Lists** - Family-wide task management
- **Calendar** - Shared family calendar with color coding

## Installation

After adding via HACS:

1. Restart Home Assistant
2. Go to **Settings** → **Devices & Services** → **Add Integration**
3. Search for "FamDo" and configure
4. Access via the "FamDo" sidebar item
